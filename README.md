# robosys2025

ロボットシステム学の課題1のプログラムをまとめたリポジトリです。
標準入力を利用した簡単な計算プログラムや、文章の雰囲気を数値化するデモプログラムなどが含まれています。

## ■ このソフトウェアでできること

### ● plus  
標準入力から渡された数値をすべて加算して出力します。
例: echo -e "1\n2\n3" | ./plus
結果:6

---

### ● moodcalc  
文章中の単語をもとに、簡易的な「ポジティブ / ネガティブ」度を計算するデモプログラムです。  
（授業の演習として作成した簡易版であり、本格的な自然言語処理ではありません。）

例：echo "今日はとても楽しかった" | ./moodcalc

---

### ● test.bash  
上記プログラムが正しく動作するかを GitHub Actions 上で自動テストするためのスクリプトです。  
ローカルでも次のように実行できます：./test.bash

---

## ■ 自動テスト（GitHub Actions）の結果

本リポジトリでは `.github/workflows/test.yml` により、  
`plus` および `moodcalc` の動作確認を自動で行う設定になっています。

テスト内容：
- plus が正しく数値を足し算できるか  
- moodcalc が特定の入力で想定どおりの数値を返すか  
- スクリプトが正常に実行できるか  

テストは push または pull request を行うと自動で実行されます。
